# עבודת קיץ אינטראקטיבית בכימיה — כיתה י״א

קובץ זה מלווה את דף העבודה האינטראקטיבי בכימיה לתלמידי כיתה י״א.


## מטרת הדף

דף העבודה נועד לחזרה עצמאית על נושאים מרכזיים בכימיה לקראת כיתה י״ב. הדף כולל שאלות רבות־ברירה, שאלות פתוחות, כיווני חשיבה, מענה להשוואה, בדיקה עצמית וסיכום למורה.

הדף אינו בודק את כל התשובות באופן אוטומטי. בשאלות פתוחות התלמידים מתבקשים לכתוב תשובה, להשוות למענה, לבצע בדיקה עצמית ולהעריך את תשובתם.

## הנחיות לתלמידים

1. פתחו את דף העבודה בדפדפן.
2. עברו על הסעיפים לפי הסדר.
3. בשאלות פתוחות כתבו תשובה לפני פתיחת כיוון החשיבה או המענה להשוואה.
4. לאחר הצגת המענה, בצעו את הבדיקה העצמית וסמנו את מצב התשובה שלכם.
5. בסיום העבודה עברו ללשונית **סיכום למורה**.
6. לחצו על **רענן סיכום**.
7. העתיקו את הדוח והגישו אותו לפי הנחיות המורה.

חשוב: התשובות נשמרות בדפדפן שבו עבדתם. אם תעברו למחשב אחר או לדפדפן אחר, ייתכן שהתשובות לא יופיעו שם.

## פרסום באמצעות GitHub Pages

1. העלו למאגר את הקובץ `index.html`.
2. היכנסו ל־**Settings** של המאגר.
3. עברו ל־**Pages**.
4. בחרו פרסום מתוך הענף `main` ומתיקיית `/root`.
5. שמרו את ההגדרות.
6. לאחר כמה דקות יופיע קישור לדף העבודה.

את הקישור שיתקבל אפשר לשלוח לתלמידים.

## פרטיות ושמירת נתונים

הדף שומר את תשובות התלמידים באופן מקומי בדפדפן, באמצעות `localStorage`. הוא אינו שולח תשובות לשרת ואינו אוסף נתונים באופן אוטומטי.

לכן, כדי שהמורה יקבל את התשובות, התלמידים צריכים להעתיק את **סיכום למורה** ולהגיש אותו.

אין להכניס לדף או למאגר שמות תלמידים, מספרי זהות או מידע אישי אחר.

## עדכון הדף

כדי לעדכן את דף העבודה:

1. ערכו את קובץ ה־HTML במחשב.
2. שמרו אותו בשם `index.html`.
3. העלו את הקובץ המעודכן ל־GitHub במקום הקובץ הקודם.
4. בצעו Commit לשינוי.
5. המתינו כמה דקות עד ש־GitHub Pages יתעדכן.

אם התלמידים רואים גרסה ישנה, כדאי לבקש מהם לרענן את הדף או לפתוח אותו בחלון גלישה חדש.
